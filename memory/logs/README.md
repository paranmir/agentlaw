# Memory Logs

This directory records dated session events: decisions, verifications, corrections, and handoff notes. Logs are append-oriented historical records, not retrievable as current truth without confirmation against active sources.

Layout: one file per day under a year-month subdirectory, e.g. `2026-04/2026-04-27.md`. Entry format (heading, id, kind, scope, status, timestamps, body) is defined in `docs/law/MEMORY_AND_CONTINUITY_RULES.md` § Log File.
