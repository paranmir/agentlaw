# Known Facts

This directory holds remembered observations about the current project state. Each fact lives in its own Markdown file with the front matter schema and minimum format defined in `docs/law/MEMORY_AND_CONTINUITY_RULES.md` § Memory Types and § Known Fact File.

Known facts are memory records, not governing rules. They lose to higher-authority sources (constitution, law layer, current explicit instruction) and to current source / repository state when the two disagree.
