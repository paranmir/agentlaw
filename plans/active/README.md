# Active Plans

This directory holds active plans that govern in-flight non-trivial or high-risk work. Each active plan must carry the preflight fields enumerated in `docs/law/REPOSITORY_ARTIFACT_RULES.md` § Active Plan Preflight Fields (scope, affected surfaces, public contract impact, oracle, install impact, rollback path, non-goals, and so on).

When a plan lands, move its file to `plans/completed/`. The plan body remains intact for future reference; only its location changes.
