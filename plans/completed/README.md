# Completed Plans

This directory keeps completed plans as historical context for future agents. They are not retrieved as current state — the working set (`memory/working-set.md`) and active plans under `plans/active/` are the current sources.

Completed plans remain valuable as the durable record of why a change was made, what alternatives were considered, and what verification was used. Future agents may consult them when investigating prior decisions or when an issue resurfaces in a related area.
